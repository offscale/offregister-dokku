offregister_dokku
=================

[Dokku](http://dokku.viewdocs.io/dokku) is a tiny PaaS written in Bash, and leveraging Docker.

This package follows the [offregister](https://github.com/offscale/offregister) specification to setup and serve Mesos.

## Install dependencies

    pip install -r requirements.txt

## Install package

    pip install .
